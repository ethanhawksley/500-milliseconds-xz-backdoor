# Chapter 9

World's reaction to the discovery.

[CVE-2024-3094](https://nvd.nist.gov/vuln/detail/cve-2024-3094)

## 2024-03-28 (still)

Debian and distros@openwall notified

CVE assigned.


Debian rolls back 5.6.1 in 2024-03-28.

Arch Linux changes 5.6.1 to build from Git.

## 2024-03-29

Andres Freund posts warning to oss-security@openwall

RedHat announces backdoored xz shipped in Fedora Rawhide and Fedora Linux 40 beta.

## 2024-03-30

Debian shuts down builds to rebuild their build machines using Debian stable (in case the malware xz escaped the sandbox).

Haiku OS moves to GitHub source repo snapshots.
